# Epoch-Website

College tech club website built with **React**, **Vite**, **Tailwind CSS**, **GSAP**, **Framer Motion**, **Swiper**, and **React Three Fiber**.

The Vite app lives in this **`Epoch-Website/`** folder at the repository root (alongside `.git`).

## Run locally

From the **repository root**:

```bash
cd Epoch-Website
npm install
npm run dev
```

## Build

```bash
cd Epoch-Website
npm run build
```

If your shell is already in **`Epoch-Website/`**, run `npm install` / `npm run dev` / `npm run build` directly.

Static assets use the `base` path in `vite.config.js` (GitHub Pages).
